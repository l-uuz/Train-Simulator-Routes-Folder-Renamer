# Train Simulator Routes Folder Renamer

This is a tool for players and add-on developers of the game [_Train Simulator Classic_](https://store.steampowered.com/app/24010/Train_Simulator_Classic/) by [Dovetail Games](https://dovetailgames.com/). 

The folder structure in `RailWorks/Content/Routes` consists of cryptic combinations of numbers and characters, such as `00000001-0000-00000000-00000000`. This tool allows you to display the names in Windows Explorer according to the route names (e. g. `Great Western Mainline`) without affecting the game’s behavior. It is necessary to execute “Extract Route Properties” in [TS-Tools](https://rail-sim.de/forum/filebase/entry/8996-ts-tools/) before starting the tool.

The application was developed using [Microsoft Visual Studio 2026](https://visualstudio.microsoft.com/de/) and requires the [.NET 10 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/10.0/runtime).
